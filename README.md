# DevicePpi

This library provides run-time detection of an iOS device's PPI.

It is useful for applications that display content in true size, such as on-screen rulers or the 1:1 zoom feature in Print to Size.

## Example

To run the example project, clone the repo, and run `pod install` from the Example directory first.

## Requirements

iOS 8 or later.

## Installation

DevicePpi can be added to project via [CocoaPods](https://cocoapods.org).

```ruby
pod 'DevicePpi'
```
